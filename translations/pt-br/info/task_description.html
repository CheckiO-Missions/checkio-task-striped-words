Nossos robôs estão sempre trabalhando para melhorar suas habilidades linguísticas. Para esta missão, eles estão pesquisando o alfabeto latim e suas aplicações.

O alfabeto contém tanto vogais como consoantes (sim, nós dividimos as letras).
Vogais -- A E I O U Y
Consoantes -- B C D F G H J K L M N P Q R S T V W X Z

Será fornecido a você um bloco de texto com palavras diferentes. Essas palavras estão separadas por espaços em branco e diferentes pontuações. Números não são considerados palavras nessa missão (uma mistura de letras e dígitos também não formam uma palavra). Você deve contar o número de palavras (palavras listradas) nas quais as vogais e consoantes se alternam, ou seja: as palavras que você contar não podem ter duas vogais ou consoantes consecutivas. As palavras formadas por apenas uma letra não são 'listradas' -- não conte essas. As letras serem maiúsculas ou minúsculas não é algo importante para essa missão.

Entrada: Um texto como uma string. (unicode)

Saída: A quantidade de palavras 'listradas' na forma de um inteiro (int).

Exemplo: 

checkio("My name is ...") == 3
checkio("Hello world") == 0
checkio("A quantity of striped words.") == 1, "Only of"
checkio("Dog,cat,mouse,bird.Human.") == 3

Como é utilizado: A idéia nessa tarefa é um exercício útil para pesquisa e análise linguística. O processamento de texto é uma das principais ferramentas utilizadas na análise de vários livros e linguagens, além disso pode ajudar a traduzir um texto impresso para o formato digital.

Condições prévias:O texto contém apenas símbolos no padrão ASCII.
0 < len(text) < 105
